## 📏 Units Explained (Simple Roman Urdu)

### 🔹 Length (Lambai)

* Meter (m) → basic unit
* Kilometer (km) → 1000 meter
* Centimeter (cm) → 1 meter = 100 cm

📌 Example:
Road = km
Room = meter
Pencil = cm

---

### 🔹 Weight (Wazan)

* Kilogram (kg) → heavy cheez
* Gram (g) → choti cheez
* Pound (lb) → foreign system

📌 Example:
Aata = kg
Cheeni = gram

---

### 🔹 Temperature (Hararat)

* Celsius (°C) → Pakistan system
* Fahrenheit (°F) → USA system

📌 Example:
Weather app = Celsius
Fever = temperature
